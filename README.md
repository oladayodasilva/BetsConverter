# BetsConverter

BetsConverter is an AI-powered betting calculator and booking-code mapping platform.

## Features

- Odds Converter
- Profit Calculator
- Accumulator Calculator
- Cashout Calculator
- Arbitrage Calculator
- Manual Slip Converter
- Booking Code Mapping Prototype
- AI Betting Assistant
- User Accounts
- Saved Calculations
- Verification Dashboard
- SEO Blog and Glossary Pages

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- NextAuth
- OpenAI API
- Vercel

## Setup

```bash
npm install
npx prisma generate
